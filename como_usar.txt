Observações: 
O algoritmo respeita a hierarquia das operações, então, se há partes que devem ser calculadas primeiro, use parenteses 
	ex: 1 / x+1 -> 1 / (x+1)

Como usar: 
	Abra o arquivo index.html usando um navegador, ou acesse o link: 

Exemplos de funções:

polinomiais 
x**2 + 2x + 2
-x**2 - 2x - 2
-x**3 + 2 
mod(x) - módulo de x

exponenciais 
2**x
2**(-x)
2**(2x)
-2**(2x)
x**4 - 3x**3
3 * 2**(-x)

logaritmicas
ln(x) - log base e
log2(x) - log base 2
log10(x) - log base 10

trigonometricas 
sen(x)
cos(x)
tg(x)
senh(x) - seno hiperbolico
cosh(x) - cosseno hiperbolico 
tgh(x) - tg hiperbolica 

2sen(x+1)+1
-sen(x) + cos(2x)
-sen(x) + 2cos(-2x) + 3

racionais
1 / x
1 / (x-1)
(x**2-1)/(x+1)

raiz
raiz2(x) - raiz quadrada 
raiz3(x) - raiz cúbica


junções
sen(x) + 1/x
sen(x**2) + 1/x
x*sen(x)
